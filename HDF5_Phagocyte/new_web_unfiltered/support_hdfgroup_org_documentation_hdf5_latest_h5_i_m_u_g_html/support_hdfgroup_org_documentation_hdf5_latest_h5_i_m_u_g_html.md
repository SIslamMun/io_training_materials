#  HDF5 Images
##  Introduction
The HDF5 Image API (H5IM) provides functions for storing and retrieving image data in HDF5 files. The API supports common image formats and includes functions for working with image attributes such as palettes. 

See also
     [HDF5 Images API (H5IM)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html) Reference Manual
##  Image Operations
Create images with [H5IMmake_image_8bit](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#gaed2cb8f411b58e6e48d596bba1d3b59b "Creates and writes an image.") or [H5IMmake_image_24bit](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#ga151b78eea0bff9872af78f950c93c862 "Creates and writes a true color image."), and read them with [H5IMread_image](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#gaec44462e614bb5b93d5776ca27491488 "Reads image data from disk."). Check if a dataset is an image using [H5IMis_image](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#gadd8c46c44a11fa36116a0b7314f248a8 "Inquires if a dataset is an image."). Get image information with [H5IMget_image_info](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#gad1d25d015457bfa2f00faf0952cc1f75 "Gets information about an image dataset \(dimensions, interlace mode and number of associated palettes..."). Work with palettes using [H5IMmake_palette](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#gad6811d80238b25851cbde0475537b7d3 "Creates and writes a palette."), [H5IMlink_palette](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#gaa653878da63058bc7435f5ae7da8dc94 "Attaches a palette to an image."), [H5IMget_palette_info](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#ga2c669d6a99cd1f3ea0d5005cd106f5dc "Gets information about a palette dataset \(dimensions\)."), and [H5IMget_palette](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html#ga89882863675a88899c327026237aecef "Gets the palette dataset.").
