Errors and Exceptions
The class [HDF5Exception](https://support.hdfgroup.org/documentation/hdf5/latest/classhdf_1_1hdf5lib_1_1exceptions_1_1_h_d_f5_exception.html) returns errors from the Java HDF5 Interface.
Two sub-classes of [HDF5Exception](https://support.hdfgroup.org/documentation/hdf5/latest/classhdf_1_1hdf5lib_1_1exceptions_1_1_h_d_f5_exception.html) are defined: 
  1. [HDF5LibraryException](https://support.hdfgroup.org/documentation/hdf5/latest/classhdf_1_1hdf5lib_1_1exceptions_1_1_h_d_f5_library_exception.html) – errors raised by the HDF5 library code 
  2. [HDF5JavaException](https://support.hdfgroup.org/documentation/hdf5/latest/classhdf_1_1hdf5lib_1_1exceptions_1_1_h_d_f5_java_exception.html) – errors raised by the HDF5 Java wrapper code 


These exceptions are sub-classed to represent specific error conditions, as needed. In particular, [HDF5LibraryException](https://support.hdfgroup.org/documentation/hdf5/latest/classhdf_1_1hdf5lib_1_1exceptions_1_1_h_d_f5_library_exception.html) has a sub-class for each major error code returned by the HDF5 library.
