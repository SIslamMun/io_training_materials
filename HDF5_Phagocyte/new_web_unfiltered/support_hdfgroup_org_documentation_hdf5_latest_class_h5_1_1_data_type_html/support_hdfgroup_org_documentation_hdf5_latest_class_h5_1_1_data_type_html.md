## Detailed Description
Class [DataType](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_data_type.html "Class DataType provides generic operations on HDF5 datatypes.") provides generic operations on HDF5 datatypes. 
[DataType](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_data_type.html "Class DataType provides generic operations on HDF5 datatypes.") inherits from [H5Object](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_h5_object.html "Class H5Object is a bridge between H5Location and DataSet, DataType, and Group.") because a named datatype is an HDF5 object and is a base class of [ArrayType](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_array_type.html "Class ArrayType inherits from DataType and provides wrappers for the HDF5's Array Datatypes."), [AtomType](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_atom_type.html "AtomType is a base class, inherited by IntType, FloatType, StrType, and PredType."), [CompType](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_comp_type.html "CompType is a derivative of a DataType and operates on HDF5 compound datatypes."), [EnumType](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_enum_type.html "EnumType is a derivative of a DataType and operates on HDF5 enum datatypes."), and [VarLenType](https://support.hdfgroup.org/documentation/hdf5/latest/class_h5_1_1_var_len_type.html "VarLenType is a derivative of a DataType and operates on HDF5 Variable-length Datatypes.").
