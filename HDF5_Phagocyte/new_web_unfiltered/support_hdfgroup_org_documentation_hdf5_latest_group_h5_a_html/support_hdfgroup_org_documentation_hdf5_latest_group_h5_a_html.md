Attributes (H5A)
## Detailed Description
An HDF5 attribute is a small metadata object describing the nature and/or intended usage of a primary data object. A primary data object may be a dataset, group, or committed datatype. 

See also
     [HDF5 Attributes](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_a__u_g.html#sec_attribute)

    Returns a non-negative value if successful; otherwise, returns a negative value. [H5Aiterate_by_name()](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_a.html#ga75db973d69b61f673f5cdf21ac624cef "Calls user-defined function for each attribute on an object.") iterates over the attributes attached to the dataset or group specified with `loc_id` and `obj_name`. The order of the iteration and the attributes iterated over are specified by three parameters: the index type, `idx_type`; the order in which the index is to be traversed, `order`; and the attribute's position in the index, `idx`. The next attribute to be operated on is specified by `idx`, a position in the index.
For example, if `idx_type`, `order`, and `idx` are set to [H5_INDEX_NAME](https://support.hdfgroup.org/documentation/hdf5/latest/_h5public_8h.html#a8203c78e10ab2d89d8bce688a31afad3a644e6701706be4d37660864336c7bd3e), [H5_ITER_INC](https://support.hdfgroup.org/documentation/hdf5/latest/_h5public_8h.html#a6a6ddd1504d1ed61939d46d91d9441b9a80c3e083c0a77063b1a66553decfcb08), and 5, respectively, the attribute in question is the fifth attribute from the beginning of the alphanumeric index of attribute names. Warning
