## Detailed Description
Buffer to store heap ID and index
This needs to be large enough to store largest [haddr_t](https://support.hdfgroup.org/documentation/hdf5/latest/_h5public_8h.html#a8c86e866f40d7167cf9a1934c72b856f) in a worst case machine (8 bytes currently) plus an int. 

**[Deprecated](https://support.hdfgroup.org/documentation/hdf5/latest/deprecated.html#_deprecated000089)**
    Dataset region reference type that is used with deprecated reference APIs. 

Note
    This type can only be used with the "native" HDF5 VOL connector.
