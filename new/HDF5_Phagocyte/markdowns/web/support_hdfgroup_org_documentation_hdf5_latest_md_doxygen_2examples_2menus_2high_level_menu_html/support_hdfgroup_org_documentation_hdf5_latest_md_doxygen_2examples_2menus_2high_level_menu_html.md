High-level Library
  
The high-level HDF5 library includes several sets of convenience and standard-use APIs to facilitate common HDF5 operations.
  * [HDF5 Lite APIs (H5LT,H5LD)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_l_t.html)   
Functions to simplify creating and manipulating datasets, attributes and other features
  * [HDF5 Images API (H5IM)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_i_m.html)   
Creating and manipulating HDF5 datasets intended to be interpreted as images
  * [HDF5 Table APIs (H5TB)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_t_b.html)   
Creating and manipulating HDF5 datasets intended to be interpreted as tables
  * [HDF5 Packet Table APIs (H5PT)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_p_t.html)   
Creating and manipulating HDF5 datasets to support append- and read-only operations on table data
  * [HDF5 Dimension Scales APIs (H5DS)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_d_s.html)   
Creating and manipulating HDF5 datasets that are associated with the dimension of another HDF5 dataset
  * [HDF5 Optimizations APIs (H5DO)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_d_o.html)   
Bypassing default HDF5 behavior in order to optimize for specific use cases
  * [Extensions (H5LR, H5LT)](https://support.hdfgroup.org/documentation/hdf5/latest/group___h5_l_r.html)
