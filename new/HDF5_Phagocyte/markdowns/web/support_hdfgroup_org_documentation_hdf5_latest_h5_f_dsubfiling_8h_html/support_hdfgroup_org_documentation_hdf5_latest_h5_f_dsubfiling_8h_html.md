##  [◆ ](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a70a3432b7e53fba2eaf36e072540c44c)H5FD_SUBFILING_CONFIG_FILE_PREFIX
#define H5FD_SUBFILING_CONFIG_FILE_PREFIX "H5FD_SUBFILING_CONFIG_FILE_PREFIX"  
---  
Macro for name of the environment variable that specifies a prefix to apply to the subfiling configuration filename. Useful for cases where the application wants to place the configuration file in a different directory than the default of putting it alongside the generated subfiles. For example, when writing to node-local storage one may wish to place the configuration file on a scratch file system readable by all nodes, while the subfiles are initially written to the node-local storage.
The value set for this environment variable is interpreted as a pathname that must already exist.
NOTE: As this prefix string will be encoded in the driver info message that gets written to the file, there is an upper limit of about ~900 single-byte characters for this string, though possibly less due to other information the driver may encode. Avoid long prefix names where possible.

##  [◆ ](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a8350a509d56b739b6cc8f4f1934d0e00)H5FD_SUBFILING_IOC_SELECTION_CRITERIA
#define H5FD_SUBFILING_IOC_SELECTION_CRITERIA "H5FD_SUBFILING_IOC_SELECTION_CRITERIA"  
---  
Macro for name of the environment variable that provides information for selection MPI ranks as I/O concentrators
The value set for this environment variable is interpreted differently, depending on the IOC selection method chosen.
For [SELECT_IOC_ONE_PER_NODE](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a2bcf2d531a0668895308692b0c1108d7a56e4ced7a9051ec9b013d494cbfdb41e), this value is ignored.
For [SELECT_IOC_EVERY_NTH_RANK](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a2bcf2d531a0668895308692b0c1108d7a9b804064468c7e924784eebad6c1b264), this value is interpreted as a long value and must be > 0. The value will correspond to the `N` value when selecting every `N`-th MPI rank as an I/O concentrator.
For [SELECT_IOC_WITH_CONFIG](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a2bcf2d531a0668895308692b0c1108d7a5b5ece0b4617c68b355cffaf3f937b3f), this value is ignored as that particular IOC selection method is not currently supported.
For [SELECT_IOC_TOTAL](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a2bcf2d531a0668895308692b0c1108d7a5d857c5db085740ef2f620cb8bafc387), this value is interpreted as a long value and must be > 0. The value will correspond to the total number of I/O concentrators to be used.

Macro for name of the environment variable that specifies the size (in bytes) for data stripes in subfiles
The value set for this environment variable is interpreted as a long long value and must be > 0.

enum [H5FD_subfiling_ioc_select_t](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a2bcf2d531a0668895308692b0c1108d7)

This enum defines the various constants to allow different allocations of MPI ranks as I/O concentrators. 
Enumerator  
---  
SELECT_IOC_ONE_PER_NODE  |  Default selection method. One MPI rank per node is used as an I/O concentrator. If this selection method is used, the number of I/O concentrators per node can be adjusted with the [H5FD_SUBFILING_IOC_PER_NODE](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a5a5883d4d3c713fdbefd27971b8a67b7) environment variable.   
SELECT_IOC_EVERY_NTH_RANK  |  Starting with MPI rank 0, a stride of 'N' is applied to the MPI rank values to determine the next I/O concentrator. The [H5FD_SUBFILING_IOC_SELECTION_CRITERIA](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a8350a509d56b739b6cc8f4f1934d0e00) environment variable must be set to the value desired for 'N'.   
SELECT_IOC_WITH_CONFIG  |  Currently unsupported. Use a configuration file to determine the mapping from MPI ranks to I/O concentrators. The [H5FD_SUBFILING_IOC_SELECTION_CRITERIA](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a8350a509d56b739b6cc8f4f1934d0e00) environment variable must be set to the path to the configuration file.   
SELECT_IOC_TOTAL  |  Specifies that a total of 'N' I/O concentrators should be used. Starting with MPI rank 0, a stride of 'MPI comm size' / 'N' is applied to the MPI rank values to determine the next I/O concentrator. The [H5FD_SUBFILING_IOC_SELECTION_CRITERIA](https://support.hdfgroup.org/documentation/hdf5/latest/_h5_f_dsubfiling_8h.html#a8350a509d56b739b6cc8f4f1934d0e00) environment variable must be set to the value desired for 'N'.   
ioc_selection_options  |  Unused. Sentinel value
